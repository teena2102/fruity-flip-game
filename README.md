# 🍓 Fruity Flip Game

**Fruity Flip Game** is a browser-based memory card game where players flip cards to match fruit pairs. The game challenges users to improve their **memory, focus, and pattern recognition** through an engaging and interactive UI.

## 🧩 Problem Statement

Many casual games lack a balance between **simplicity and cognitive engagement**. Users often look for quick games that are:

* Easy to understand
* Mentally stimulating
* Visually engaging

**Fruity Flip Game** addresses this by combining **minimal design with memory-based gameplay mechanics**.


## 💡 Solution

A lightweight web game that:

* Uses **card-flipping logic**
* Tracks **user score dynamically**
* Encourages **memory retention and quick thinking**

Similar fruit-matching games are widely used to improve **memory and concentration skills**, especially in casual and educational contexts ([Chrome-Stats][1]).


## 🚀 Features

* 🍉 Fruit-themed interactive cards
* 🧠 Memory-based gameplay logic
* 🎯 Score tracking system
* 🔄 Smooth flip animations
* 📱 Fully responsive design
* ⚡ Fast and optimized performance


## 🎮 Gameplay

1. Cards are placed face down.
2. Player flips two cards at a time.
3. If cards match:

   * They remain visible
   * Score increases
4. If not:

   * Cards flip back
5. Game ends when all pairs are matched.
