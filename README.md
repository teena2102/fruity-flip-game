# 🍓 Fruity Flip Game

**Fruity Flip Game** is an interactive memory-based card matching game where players flip cards to find matching fruit pairs. The game is designed to enhance memory, concentration, and cognitive skills through a simple yet engaging gameplay experience.


## 🚀 Features

* 🍉 Visually appealing fruit-themed cards
* 🧠 Improves memory and focus
* 🎯 Real-time score tracking
* 🔄 Smooth card flip interactions
* 📱 Responsive design (mobile & desktop friendly)
* ⚡ Lightweight and fast


## 🎮 How to Play

1. All cards are placed face down at the start.
2. Flip any two cards by clicking on them.
3. If the cards match:

   * They remain visible
   * Your score increases
4. If the cards do not match:

   * They flip back after a short delay
5. Continue until all pairs are matched.
